# 💡 IAP — Innovation Assessment Platform

مقياس الابتكار | أداة تقييم وتحليل القدرات الابتكارية

## 🚀 تشغيل المشروع

```bash
cd IAP
python server.py
```

ثم افتح المتصفح على:
```
http://localhost:8090
```

## 📁 هيكل المشروع

```
IAP/
├── index.html      ← واجهة المستخدم (SPA)
├── server.py       ← HTTP Server + REST API
├── database.py     ← قاعدة بيانات SQLite
├── config.json     ← الأسئلة والمهارات والتوصيات (مركزي)
├── iap.db          ← قاعدة البيانات (تُنشأ تلقائياً)
└── README.md       ← هذا الملف
```

## 🌐 API Endpoints

| Method | Endpoint       | الوصف                     |
|--------|---------------|---------------------------|
| GET    | /api/questions | جلب الأسئلة               |
| POST   | /api/submit    | إرسال التقييم والحصول على النتيجة |
| GET    | /api/stats     | إحصائيات عامة             |
| GET    | /api/attempts  | جلب المحاولات السابقة       |
| DELETE | /api/attempts/:id  | حذف محاولة (يتطلب مصادقة)  |
| DELETE | /api/attempts/all  | حذف الكل (يتطلب مصادقة)    |

## ⚙️ المتطلبات

- Python 3.10+ (بدون مكتبات خارجية)
- متصفح حديث

## 📊 آلية التقييم

- **5 مواقف** تفاعلية، كل موقف يقيس مهارة
- **3 خيارات** لكل سؤال: أ=1، ب=2، ج=3
- **المجموع من 15** يحدد اللقب الابتكاري:
  - 5–8 = **المنظم**
  - 9–12 = **العملي**
  - 13–15 = **المبتكر**
- **أقل مهارة** = أولوية التطوير

## 🗺️ المراحل

### ✅ المرحلة الأولى (MVP)
- صفحة تعريف
